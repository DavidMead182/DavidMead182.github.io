Overview

This project is a personal website designed to showcase my CV/Resume and various projects. The primary problem it solves is providing a centralized platform where potential employers, collaborators, and other interested parties can easily access my professional information and project portfolio.

Problem Solved:
Before creating this website, my CV and project details were scattered across different platforms and formats, making it difficult for others to get a comprehensive view of my skills and experience. This website consolidates all that information in one place, making it more accessible and easier to navigate.

Challenges Faced:
One of the main challenges was ensuring the website was fully responsive and looked good on all devices. This was overcome by using Tailwind CSS, which provides a robust set of responsive design utilities. Another challenge was managing the dynamic content for the project dashboard. This was addressed by using JavaScript to load project data dynamically, making it easier to update and maintain.

Overall, this project has been a valuable addition to my professional toolkit, helping me present my skills and experience in a clear and organised way.

Features

  • Responsive Design: The website is fully responsive, ensuring a great experience on all devices, from desktops to mobile phones.

  • Project Dashboard: A dedicated section to showcase all my projects with descriptions, technologies used, and links to repositories or live demos.

  • Dynamic Content: Projects are dynamically loaded, making it easy to update and maintain the site.

  • Modern Styling: Utilises Tailwind CSS for a clean and modern design, with custom animations and transitions.
